# Reporting Guide

Use this to report misuse of the People First Design badge or violations of the principles.

## What to report
- False “Verified” claims or missing badge IDs
- Misleading evidence pages
- Dark patterns, privacy issues, or abusive UX
- Security or data handling concerns

## What to include
- URL(s) and screenshots
- Claimed badge state and ID (if any)
- Brief description of the issue
- Your contact (optional, for follow-up)

## How to report
1. Open an issue in this repository with the details above.  
    - If sensitive, mark it clearly.
2. If you cannot use issues, email the maintainers with the same information.

## What to expect
- We will review, request clarification if needed, and document outcomes.
- Verified badges may be updated, revoked, or left unchanged based on findings.
- Non-Verified misuse will be noted for community awareness.
